如何存款?
登入帳號 → 點擊網站首頁｢我的｣ →  點擊｢取款｣按鈕 → 進入存款頁面
1. 第一次存款先綁定銀行卡，綁定銀行卡步驟：
登入帳號 → 點擊網站首頁｢我的｣ → 點擊｢我的錢包｣ → 添加銀行卡
2. 綁定完成後選取存款方式:
選取或輸入存款金額 → 點擊｢立即存款｣ → 完成存款動作
